Código do Municipio (IBGE)
Além do conhecido CEP, os municipios também recebem um código pelo IBGE.

Este código, chamado de Código do Municipio é composto por 7 dígitos, sendo os dois primeiros referentes ao código da Unidade da Federação.

Para consultar o código do seu municipio basta acessar o site do ibge:

https://www.ibge.gov.br/explica/codigos-dos-municipios.php
