GTIN
O GTIN, sigla de “Global Trade Item Number” é um identificador para itens comerciais desenvolvido e controlado pela GS1, antiga EAN/UCC.

O GTIN é utilizado para recuperar informação pré-definida e abrange desde as matérias primas até produtos acabados.

GTIN é um termo “guarda-chuva” para descrever toda a família de identificação das estruturas de dados GS1 para itens comerciais (produtos e serviços).

Observação: Preencher com o código GTIN-8, GTIN-12, GTIN-13 ou GTIn-14, não iformar o conteúdo da TAG em caso de o produto não possuir este código.
