Código de Situação da Operação – Simples Nacional
O Código de Situação da Operação do Simples Nacional (CSOSN) trata-se de uma lista enumerada de operações de uma empresa, que estabelece critérios tributários para cada situação, quando esta é classificada no regime do Simples Nacional.

102= Tributada pelo Simples Nacional sem permissão de crédito.

103= Isenção do ICMS no Simples Nacional para faixa de receita bruta.

300= Imune.

400= Não tributada pelo Simples Nacional.
