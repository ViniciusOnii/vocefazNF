GTIN do produto
O GTIN, sigla para Global Trade Item Number, é o número de identificação global de um item comercial. Ele é único para cada produto, sendo formado por uma estrutura numérica a partir da qual é gerado o código de barras.

Observação: Preencher com o código GTIN-8, GTIN-12, GTIN-13 ou GTIN14 (antigos códigos EAN, UPC e DUN-14) da unidade tributável do produto, não informar o conteúdo da TAG em caso de o produto não possuir este código.
