Finalidade da Emissão NF-e
A finalidade de emissão é utilizada para informar qual tipo de operação vai ser feita na nota e para o recolhimento de impostos. Verifique abaixo as existentes.

NFe normal: NFe que não tem nenhuma finalidade específica. É utilizada para realizar NF-e de venda, Remessa de mercadoria, Bonificação, entre outras.

NFe complementar: NFe complementar é utilizada para acrescentar valores de imposto que não foram informados na NF-e original. Ex: Foi feita a emissão de uma NF-e, esta deveria ter ICMS destacado, porém na emissão não foi informado o ICMS. Neste caso, é feita uma NF-e complementar, apenas para informar os valores de ICMS corretos da nota.

NFe de ajuste: A NFe de Ajuste serve para ser utilizada em operações sem circulação de produtos, como uma NF de transferência de crédito, crédito de ativo permanente e etc.

NFe de devolução de mercadoria: NFe de devolução de mercadoria é utilizada quando for emitir uma NF-e de devolução de venda ou de compra. Neste caso, deve-se utilizar a finalidade 4 - Devolução de Mercadoria.
